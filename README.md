Архитектурные жемчужины Витебска
Данный проект представляет собой адаптивный одностраничный веб-сайт (landing page), разработанный в рамках технического задания по теме культурного наследия Витебского региона. Сайт посвящен архитектурным достопримечательностям города, их истории и визуальному облику.

Структура проекта
Файловая система организована в соответствии с требованиями технической документации:

index.html — основной файл лендинга, содержащий всю семантическую разметку.

style.css — файл основных стилей проекта.

media.css — файл с медиа-запросами для обеспечения адаптивности.

/js/ — папка с JavaScript файлами (script.js, media.js, youtube.js, wow.min.js).

/img/ — папка с оптимизированными графическими ресурсами (логотипы, фотографии архитектуры).

/audio/ — папка с аудиоматериалами (фольклорные мелодии).

/video/ — папка для видеоконтента (если используется локально).

/fonts/ — локальные шрифты проекта.

Особенности реализации
Технологический стек
HTML5: Использование семантических тегов (header, nav, main, section, footer) для корректной структуры и SEO.

CSS3: Применение CSS-переменных для унификации цветовой гаммы, Flexbox и Grid Layout для построения сеток. Все анимации и переходы реализованы с плавными таймингами.

JavaScript (ES6+): Реализация логики поиска по видео, управления модальными окнами, мобильного меню и взаимодействия с пользователем без использования тяжелых фреймворков.

Интерактивность и мультимедиа
Слайдер: Использование библиотеки Swiper.js для демонстрации фотогалереи архитектурных объектов.

Видеогалерея: Интеграция с YouTube API, позволяющая просматривать тематические ролики и выполнять поиск по контенту.

Анимация: Подключение библиотеки WOW.js для активации анимаций появления блоков при прокрутке страницы.

Аудио: Встроенный плеер с фольклорными композициями региона для создания атмосферы погружения в культуру.

Адаптивность и доступность
Сайт полностью адаптирован под экраны от 320px до 1440px с ключевыми точками перехода на 320, 690 и 1440 пикселях.

Соблюдение стандартов WCAG: обеспечена контрастность текста, добавлена навигация с клавиатуры и реализовано управление фокусом в модальных окнах (trap focus).

Для всех изображений добавлены альтернативные тексты (alt).

Инструкция по запуску
Распакуйте архив с проектом.

Убедитесь, что все ресурсы находятся в соответствующих папках (особенно /img/ и /audio/).

Откройте файл index.html в современном браузере (Chrome, Firefox или Яндекс.Браузер).

Для корректного отображения видео из YouTube требуется стабильное интернет-соединение.

Обоснование выбора темы и дизайна
Выбор темы «Архитектурные жемчужины Витебска» обусловлен богатым историческим фоном города. Визуальное решение опирается на следующие принципы:

Цветовая схема: Использование синих и белых тонов подчеркивает ассоциацию с городом над Двиной и его храмовой архитектурой.

Типографика: Шрифт Inter выбран за его современный вид и высокую читаемость, что контрастирует с изображениями старинных зданий.

Контент: Материалы подобраны таким образом, чтобы продемонстрировать преемственность поколений — от древней Ратуши до современных культурных объектов.
